# Evil Eye Filter

A RuneLite plugin for Old School RuneScape Leagues that filters the **Evil Eye** necklace right-click menu to only show the regions you have unlocked.

## The Problem

When you right-click the Evil Eye necklace, it shows every region in the game — even ones you haven't unlocked. This makes the menu cluttered and harder to use when you only have access to a few regions.

## The Solution

This plugin hides any regions you haven't unlocked from the right-click menu, keeping it clean and relevant to your current playthrough.

## Configuration

In the plugin settings panel you'll find a checkbox for every region in Leagues:

- Misthalin
- Asgarnia
- Karamja
- Kandarin
- Desert
- Morytania
- Fremennik
- Tirannwn
- Wilderness
- Great Kourend
- Varlamore

Simply **check the regions you have unlocked** and the rest will be hidden from the Evil Eye right-click menu automatically. Changes take effect immediately — no restart required.

## Notes

- This plugin only affects the Evil Eye necklace right-click menu. It does not modify the teleport interface that opens after selecting a region.
- Standard menu options (Wear, Use, Examine, Cancel, Last-destination) are always kept regardless of your settings.
- Designed for Leagues but will work with any future content that uses the same item and menu structure.
